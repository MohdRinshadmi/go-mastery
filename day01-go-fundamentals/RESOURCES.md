# Day 01 Resources — Fundamentals

Curated, topic-specific links. All real go.dev / official sources.

- **A Tour of Go — Basics** — https://go.dev/tour/basics/1
  Interactive walk through packages, variables, constants, and functions.

- **Effective Go** — https://go.dev/doc/effective_go
  The canonical idiom guide. Read the sections on names, constants, and functions.

- **Go by Example: Variables / Constants / Functions** —
  https://gobyexample.com/variables ,
  https://gobyexample.com/constants ,
  https://gobyexample.com/functions
  Tiny runnable snippets for each primitive.

- **Constants (The Go Blog)** — https://go.dev/blog/constants
  Deep dive into typed vs untyped constants and how `iota` really works.

- **Defer, Panic, and Recover (The Go Blog)** — https://go.dev/blog/defer-panic-and-recover
  How `defer` ordering and execution work.

- **Using Go Modules (The Go Blog)** — https://go.dev/blog/using-go-modules
  Modules, `go.mod`, `go.sum`, and dependency basics.

- **Go Modules Reference** — https://go.dev/ref/mod
  The full reference for module paths, versioning, and the toolchain.

- **The Go Programming Language Specification — Declarations and scope** —
  https://go.dev/ref/spec#Declarations_and_scope
  The exact rules behind shadowing, short variable declarations, and `iota`.
